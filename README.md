# BTC Market Data Analytics

This is a research project focused on BTC/USD market data analytics and reference price monitoring.

The goal of the project is to compare BTC/USD reference prices from reliable oracle/data providers with exchange market data and prediction market data.

## Use Case

The project requires low-latency BTC/USD price data, including:

- Bid price
- Ask price
- Mid price
- Timestamped price reports
- Reference price monitoring

## Data Sources of Interest

- Chainlink Data Streams BTC/USD CEX Price Stream
- Centralized exchange BTC/USD or BTC/USDT market data
- Prediction market data

## Purpose

This repository is used as a public project page for requesting access to Chainlink Data Streams and documenting BTC/USD market data research.
